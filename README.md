Vuln Scanner - Fast-profile (masscan + nmap) - Full project
=========================================================

Quick start:
1. Install Docker & Docker Compose.
2. Unzip this repo and open the folder.
3. Build & run:
   docker-compose build
   docker-compose up --scale worker=2

Frontend: http://localhost:3000
Backend OpenAPI: http://localhost:8000/docs

Important: Only scan targets you own or have permission to scan.
